# Kunal Singh Finance Portfolio

Static portfolio website for GitHub Pages.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

## GitHub Pages Setup

1. Create a new public GitHub repository named `kunal-singh-finance-site`.
2. Push these files to the repository.
3. In GitHub, open Settings -> Pages.
4. Set the source to `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save. GitHub will publish the site at:

```text
https://kunalsingh-finance.github.io/kunal-singh-finance-site/
```
